# Algoritmos de Ordenação em C++

Projeto desenvolvido para estudar e comparar diferentes algoritmos de ordenação.

## Algoritmos Implementados

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- std::sort (Biblioteca STL)

## Objetivo

Comparar diferentes técnicas de ordenação e analisar seu funcionamento e desempenho.

## Conceitos Aplicados

- Estruturas de Dados
- Recursão
- Divisão e Conquista
- Vetores (std::vector)
- Templates
- Medição de Tempo de Execução
- Algoritmos de Ordenação

## Implementações

### Bubble Sort

Realiza trocas sucessivas entre elementos adjacentes.

### Selection Sort

Seleciona o menor elemento e o posiciona corretamente.

### Insertion Sort

Insere cada elemento na posição correta da parte já ordenada.

### Merge Sort

Utiliza a estratégia de Divisão e Conquista para ordenar o vetor.

### Quick Sort

Divide o vetor em partições utilizando pivôs.
